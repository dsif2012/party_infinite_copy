# 長期成長與解鎖系統｜Party Infinite

## 設計目標

本遊戲不只是單場副本求生，也有長期玩家生涯壓力。

長期目標：

- 玩家必須逐步變強
- 雙核合作不能永遠停留在低難度
- 官方系統、論壇、市集、勢力情報都要隨通關進度解鎖
- 隊伍進度與個人履歷共同決定可見內容
- 第一年內至少要完成一次 C 級副本
- 第一次 C 級後開啟正式戰力評估

## 解鎖邏輯

新版解鎖不只看單一主角，而是同時參考：

1. 玩家 A 個人履歷
2. 玩家 B 個人履歷
3. 雙核隊伍共同通關最高難度
4. 最近副本表現
5. 是否與高階玩家或勢力產生穩定互動

實作原則：

- 只要雙核共同通關更高難度，官方副本選項、論壇分區、市集品級與勢力情報會同步升級
- 若只有個人參與過較高難度，個人會有額外視野，但未必能完全帶動整隊解鎖

## 局外資源分層

### 官方點數

官方點數是最穩定的成長資源。

用途：

- 購買技能
- 購買裝備
- 購買消耗品
- 申請副本方向
- 購買副本情報
- 鑑定或登錄道具

限制：

- 技能不是買完就精通
- 高階能力仍需訓練、實戰與條件累積

### 遊戲論壇

論壇是情報、風評、招募與謠言混雜的地方。

解鎖節奏：

- F/E：只能看低階副本心得與零散新手討論
- D：開始看見較穩定的隊伍合作帖、玩家勢力風評與交易警示
- C：開放更可信的戰力討論區、混編副本經驗、勢力觀察與高風險警示

### 玩家市集

市集是道具、媒介、情報與人脈交換入口。

解鎖節奏：

- F/E：低階消耗品、基礎裝備、散亂情報
- D：可見較穩定的媒介、委託與臨時招募
- C：可接觸更完整的高階裝備、專長委託與勢力影響下的交易

### 官方副本選項

官方副本方向會隨進度提高可控度。

- F/E：只能選低精度難度或模糊方向
- D：可開始申請混編、護送、戰鬥訓練等偏向
- C：可接觸更正式的隊伍匹配、風險摘要與部分高階評估入口

## 第一年度壓力

第一年內，玩家至少要完成一次 C 級副本。

若未完成，可能後果：

- 低難度指定自由度下降
- 系統更容易把隊伍推往 C 級
- 低階副本收益遞減
- 官方或玩家勢力開始對雙核做出評估與施壓

## 第一次 C 級後

第一次 C 級通關或有效參與完成後，開啟正式戰力評估。

正式戰力評估用途：

- 決定後續混編副本中的可信度
- 影響玩家圈對指揮權的接受度
- 影響官方、論壇與勢力對角色的判讀
- 成為後續年度門檻的重要依據

### C 級後的戰力平分

C 級之後，玩家圈與部分匹配機制會開始出現 `戰力平分` 或 `戰力攤平` 概念。

這不是把每個人的面板真的變成一樣，而是：

- 混編副本會盡量避免出現極端失衡隊伍
- 隊伍內高戰力者通常會被期待承擔更多前線或指揮責任
- 低戰力者若被高階隊伍帶入，副本壓力也會一起被拉高

結果：

- 通過 C 級後，雙核若與更高戰力玩家組隊，不能再把自己當作低階新手看待
- 玩家圈會用正式戰力評估與實績重新定位兩位玩家角色

## 每次副本後應更新

- 副本名稱
- 難度
- 結果
- 是否指定難度或方向
- 是否消耗情報 / 官方點數
- 是否引入勢力、人情債或仇恨
- 是否發生實質戰鬥
- 是否影響戰力評估
- 是否提升論壇、市集、官方副本選項或勢力接觸層級

## 初期主持提醒

- 初期不要讓白塔或風聲一口氣展開全部真相
- 白塔與風聲都應該隨通關進度逐漸露出更深部分
- 兩位玩家角色的成長要同時包含：
  - 副本內能力成長
  - 局外資源取得
  - 人際與勢力風險累積
