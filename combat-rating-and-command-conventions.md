# 綜合戰力數值與臨時指揮約定｜Party Infinite

## 定位

本檔固定多人混編副本中常見的：

- 綜合戰力數值
- 臨時指揮順位
- C 級後正式評估
- 戰力平分生態

這些不是所有副本的硬規則，但在 D 級以上玩家圈十分常見。

## 綜合戰力數值

綜合戰力數值是玩家圈常用的粗略評估，不等於官方完整面板。

通常會綜合考量：

1. 戰鬥能力
2. 生存能力
3. 機動能力
4. 防禦與承壓
5. 控場或打斷
6. 支援能力
7. 規則理解
8. 指揮能力
9. 副本實績
10. 是否完成過 C 級以上副本

## C 級前與 C 級後

- F / E / D 階段只有粗估，可信度有限
- 完成第一次 C 級後，才會開啟較正式的戰力評估
- C 級前的戰力多半是自報、隊伍內估算或勢力內部說法

因此：

- 兩位玩家角色在開局時都沒有正式戰力評估
- 他們只能報定位、短板與粗估實績

## 臨時指揮順位

D 級以上混編副本常見開場流程：

1. 報身份：固定隊 / 小隊 / 散人 / 勢力
2. 報主要定位
3. 報正式戰力；若無，說明未評估與粗估專長
4. 戰力較高者優先取得第一順位指揮承接權
5. 若高戰力者不適任總指揮，則順延給較適合的規則型、支援型或協調型角色
6. 若隊伍內有明確專長者，總指揮可分出局部判斷權

## 指揮位責任

臨時指揮通常負責：

- 路線決策
- 隊伍分工
- 戰鬥與撤退節奏
- NPC 護送與任務流程排序
- 對陌生玩家衝突進行排序

臨時指揮通常不能：

- 強制徵用私人道具
- 強制公開核心能力
- 強迫別人代承接高危物或規則責任

## 為什麼高戰力不一定是總指揮

高戰力者常是前排、爆發位或戰術執行位，不一定適合總指揮。

常見情況：

- 高戰力者先拿第一順位指揮承接權
- 他若不適合總指揮，可保留局部戰鬥主導權
- 總指揮再順延給更能處理全局的人

## C 級後的戰力平分

C 級後，玩家圈會更常提到 `戰力平分`。

這裡的戰力平分不是數值平均，而是三件事：

1. 匹配生態會盡量避免極端失衡
2. 隊伍內高戰力者會被期待多承擔
3. 低戰力者若被帶進更高階隊伍，也會一起承受更高副本壓力

因此：

- 雙核通過 C 級後，兩位玩家角色不能只靠「我們是新人」迴避責任
- 若其中一人在戰鬥評估上較高，野團會傾向先看他是否接前線指揮
- 若另一人在規則與協調評估上更穩，則總指揮可順延給更適合的人

## 雙玩家核心的初期處境

開局時：

- 兩人是固定雙人搭檔
- 沒有正式戰力評估
- 沒有高階勢力背書
- 真正優勢在互相理解與執行默契

這代表：

- 在野團中，他們會被低估
- 但若雙核開場配合穩，會比散人更快取得可信度
