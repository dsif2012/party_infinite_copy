# Party Infinite

雙玩家無限流 TRPG 長期存檔 repo。

本版本是 `infinite-prompt` 的平行新檔，保留規則求生、局外成長、隊伍經營與勢力灰度，但預設玩法改成：

- 兩位玩家各自操控自己的角色。
- 不預設角色姓名、定位與背景。
- 由玩家自建雙核心搭檔。
- 其他隊友依副本與匹配條件彈性加入。

請優先閱讀：

1. `NEW_WINDOW_PROMPT.md`
2. `current-state.md`
3. `world-rules.md`
4. `team-party-file.md`
5. `progression-system.md`
6. `combat-rating-and-command-conventions.md`
7. `runs/001-freshman-binding-run.md`

本 repo 只保存遊戲設定與劇情存檔，不應存放現實個資。
