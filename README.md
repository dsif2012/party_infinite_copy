# Party Infinite

雙玩家無限流 TRPG 長期存檔 repo。

本版本是 `infinite-prompt` 的雙玩家模板版，保留規則求生、局外成長、隊伍經營與勢力灰度，但預設玩法改成：

- 兩位玩家各自操控自己的角色。
- 兩位玩家的姓名、關係、定位與背景，在第一次遊玩時自行設定。
- 角色能力、互動細節與成長方向由玩家演繹補充。
- 其他隊友依副本與匹配條件彈性加入。
