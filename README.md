# Party Infinite

雙玩家無限流 TRPG 長期存檔 repo。

本版本是 `infinite-prompt` 的平行新檔，保留規則求生、局外成長、隊伍經營與勢力灰度，但預設玩法改成：

- 兩位玩家各自操控自己的角色。
- 角色基礎名義為 `夜讀` 與 `黎`。
- 只固定兩人是學長 / 學弟關係。
- 其他定位、背景、能力與互動細節由玩家演繹補充。
- 其他隊友依副本與匹配條件彈性加入。
