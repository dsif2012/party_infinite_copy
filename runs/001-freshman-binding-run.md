# 副本紀錄 001｜舊教學樓四樓：請一起簽退

- 難度：F
- 類型：新手引導 / 規則求生 / 雙人合作
- 結果：HE
- 全員存活：是

## 副本概念

這是夜讀與黎第一次共同進入的副本。

場域是一棟放學後仍亮著部分燈光的舊教學樓四樓。  
規則核心不是擊敗怪物，而是分清楚：

- 哪些教室可以進
- 哪些簽名不能代簽
- 什麼叫做「一起離開」

## 主要成果

- 夜讀與黎確認彼此都是真實玩家，不是副本投影
- 兩人學會基本共享資訊
- 兩人成功避開代簽、替人承接與錯誤帶路陷阱
- 完成四樓正確簽退流程
- 取得 `副本綁定道具`

## 核心解法

這次 HE 的核心不是跑得最快，而是正確完成「一起簽退」。

關鍵流程：

1. 不分頭太久，先確認彼此都還在同一套規則裡
2. 不替陌生名字代簽
3. 不替已失聯學生補簽
4. 找到真正對應兩人的簽退欄
5. 一起離開，而不是各自搶出口

## 副本綁定道具

獎勵道具：

- 名稱：`同頁扣環｜雙人副本綁定器`
- 類型：副本綁定道具
- 狀態：常駐生效

效果：

- 讓夜讀與黎在之後的副本匹配中固定進入同一個副本
- 視為副本系統承認的雙人綁定
- 不保證其他隊友配置

限制：

- 目前只對夜讀與黎兩人有效
- 不可當成市集一般裝備出售理解
- 高階副本或特殊規則是否能干擾，暫時未知

## 能力與隊伍影響

- 夜讀與黎確認彼此適合長期固定搭檔
- 角色具體定位由玩家在後續遊玩中自行定型
- 雙核雛形成立
- 之後的副本可以以穩定雙玩家合作為前提設計

## 通關後解鎖

- 官方點數：夜讀 +80，黎 +80
- 官方系統：開啟低階功能
- 遊戲論壇：開啟新手區
- 初步知道 `白塔` 與 `風聲` 兩個名字

## 主持備忘

- 這場副本的功能是建立雙玩家玩法，不是塞滿複雜戰鬥
- 綁定道具必須被演成合理獎勵，不是硬塞便利功能
- 後續副本可開始放入更多混編、勢力與戰力協調壓力
