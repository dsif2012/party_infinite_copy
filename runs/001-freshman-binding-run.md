# 副本框架 001｜舊教學樓四樓：請一起簽退

- 難度：F
- 類型：新手引導 / 規則求生 / 雙人合作
- 結果：待遊玩
- 全員存活：待判定

## 副本概念

這是兩位玩家角色第一次共同進入的副本。

場域是一棟放學後仍亮著部分燈光的舊教學樓四樓。  
規則核心不是擊敗怪物，而是分清楚：

- 哪些教室可以進
- 哪些簽名不能代簽
- 什麼叫做「一起離開」

## 主要成果

- 讓兩位玩家角色第一次一起面對副本規則
- 讓兩人建立基本共享資訊習慣
- 讓兩人理解代簽、承接與錯誤帶路陷阱
- 作為取得 `副本綁定道具` 的第一場副本

## 核心解法

這場的 HE 核心不是跑得最快，而是正確完成「一起簽退」。

關鍵流程：

1. 開局先讓玩家設定姓名與彼此關係
2. 不分頭太久，先確認彼此都還在同一套規則裡
3. 不替陌生名字代簽
4. 不替已失聯學生補簽
5. 找到真正對應兩人的簽退欄
6. 一起離開，而不是各自搶出口

## 副本綁定道具

獎勵道具：

- 名稱：`同頁扣環｜雙人副本綁定器`
- 類型：副本綁定道具
- 狀態：常駐生效

效果：

- 讓兩位玩家角色在之後的副本匹配中固定進入同一個副本
- 視為副本系統承認的雙人綁定
- 不保證其他隊友配置

限制：

- 目前只對兩位玩家角色有效
- 不可當成市集一般裝備出售理解
- 高階副本或特殊規則是否能干擾，暫時未知

## 能力與隊伍影響

- 兩位玩家角色完成第一次共同副本
- 角色具體定位由玩家在後續遊玩中自行定型
- 雙核心玩法正式成立
- 之後的副本可以以穩定雙玩家合作為前提設計

## 通關後解鎖

- 官方點數：兩位玩家角色各 +80
- 官方系統：開啟低階功能
- 遊戲論壇：開啟新手區
- 初步知道 `白塔` 與 `風聲` 兩個名字

## 通關後現實局外增強

第一場副本結束後，應進入現實休息期與局外增強教學。

若兩名玩家角色至少 HE 生還，可逐步開放：

- 遊戲官方系統：結算、官方點數、基礎購買、基礎鑑定、低精度副本方向申請。
- 遊戲論壇：新手區、低階心得、真假混雜的副本傳聞、招募訊息。
- 玩家市集：先知道入口或傳聞，不直接開放高階交易。
- 白塔：初步知道名稱與「新手互助、重流程、重秩序」的公開形象，但不判定善惡。
- 風聲：初步知道名稱或模糊傳聞，作為隱性情報圈的伏筆。
- 現實側行動：休息、討論、整理線索、規劃下一場副本、購買低階資源。

這些內容只能在第一場副本結束後使用，不應提前幫助玩家解第一場 F 級副本。

## 主持備忘

- 這場副本的功能是建立雙玩家玩法，不是塞滿複雜戰鬥
- 綁定道具必須被演成合理獎勵，不是硬塞便利功能
- 後續副本可開始放入更多混編、勢力與戰力協調壓力
